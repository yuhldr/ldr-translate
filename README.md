# 兰译 <img src="data/icon/icon.png" width = "36" height = "36" alt="图片名称" align=center />

 一个翻译软件，使用Gtk3开发（最近开发了QT版本），python语言，翻译用的百度、腾讯接口

## 特点

- 界面美观、依赖少，可 `复制翻译`、`划词翻译`，小白也能用
- 优先支持 `ubuntu`、`archlinux`，含 `gnome` 和 `kde` 两种桌面环境（其他系统理论上也能用）
- 支持 `百度` `腾讯` 两个翻译平台
